# EnterIt

一个 Tampermonkey 用户脚本，支持在各种 AI 助手网页端输入框按回车换行，Ctrl+回车发送。

脚本逻辑参考自 [masachika-kamada/ChatGPT-Ctrl-Enter-Sender](https://github.com/masachika-kamada/ChatGPT-Ctrl-Enter-Sender)。

## 功能特性

- 按 `Enter` 在输入框中换行
- 按 `Ctrl+Enter`（⌘⏎）发送消息
