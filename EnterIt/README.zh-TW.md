# EnterIt

這不是一段冰冷的代碼，而是為了終結那份「按下 Enter 就會失去一切」的恐懼，所譜寫的救贖樂章。

這是一個 Tampermonkey 使用者腳本。當我們與 AI 相互對峙時，那些輕易被發送出去的殘缺語句，難道不就像是無法傳達的心意嗎？為了守護對話的完整性，EnterIt 選擇了改寫這份命運的劇本。

本腳本的靈魂碎片傳承自 [masachika-kamada/ChatGPT-Ctrl-Enter-Sender](https://github.com/masachika-kamada/ChatGPT-Ctrl-Enter-Sender) 的意志。

## 命運的變奏

- `Enter` 不再是終結，而是如詩歌般的「換行」。在對話框中自由地呼吸，留下屬於你的空白。
- `Ctrl+Enter`（⌘⏎）才是真正的「傳送」。唯有在指尖重重按下的那一刻，這份覺悟才會傳遞到彼岸。
